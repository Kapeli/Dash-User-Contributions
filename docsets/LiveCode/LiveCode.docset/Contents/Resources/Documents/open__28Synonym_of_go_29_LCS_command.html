open (Synonym of go)

Typecommand
DictionaryLCS
Syntax
go [invisible] [to] <card> [of <stack>] [{as <mode> |in [a] new window|in <window>}]
go [invisible] [to] {first | prev[ious]| next | last | any} [marked] [<card>]
go [invisible] [to] {recent | start | finish | home} <card>
go {forward | forth | back[ward]} [<number>]
Synonymsopen
Summary

Navigates to another card or stack.

EditionCommunity
Introduced1.0
OSmac, windows, linux, ios, android
Platformsdesktop, mobile
Parameters
NameTypeDescription
card

Any card reference. Cards can be described by their name, number, or ID properties.

stack

Any stack reference, or any file path or http URL reference that resolves to a stack file. If you specify a file path or http URL, the command opens the main stack of the specified stack file.

mode
room
editable window:
palette:
modal dialog box:
modeless dialog box:
sheet dialog box: appears in
drawer: appears in , centered at left side if there's
window

The name or windowID property of any open stack. If a window is specified, the stack opens in that window, replacing the stack that was previously displayed in that window.

number

The number of cards to move within the recent cards list.

Example
go to card 2
go to stack "Controls" as palette
go to next marked card
go back 7
go invisible stack "Preferences"
local tStackFile, tStackFolder
put the effective filename of me into tStackFile
set the itemDelimiter to "/"
put item 1 to -2 of tStackFile into tStackFolder
set the defaultFolder to tStackFolder
go stack "somethingElse.livecode"
-- "somethingElse" being the name of a stack file in the same folder
Values
NameTypeDescription
The result

If the stack or card you specify doesn't exist, an error message is returned by the result function.
