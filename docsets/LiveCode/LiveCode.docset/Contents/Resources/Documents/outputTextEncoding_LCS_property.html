outputTextEncoding

Typeproperty
DictionaryLCS
Syntax
set the outputTextEncoding to <textEncoding>
Summary

The outputTextEncoding property determines what text conversion to perform when writing text strings to stdout.

EditionCommunity
Introduced4.6.3
OSmac, windows, linux
Platformsserver
Parameters
NameTypeDescription
textEncoding

The type of text encoding to use. 1252); this is the native text encoding for the Windows engine encoding of the Mac engine text encoding of the Linux engine running on; this is the default

windows-1252, windows: use the Windows 'Latin-1' encoding (codepage
macintosh, mac: use the MacRoman encoding; this is the native text
iso-8859-1, linux: use the ISO-8859-1 encoding; this is the native
utf-8, utf8: use the UTF-8 encoding
native: use the native encoding for the platform the engine is
Example
set the outputTextEncoding to "utf8"
put the outputTextEncoding into tCurrentTextEncoding
Description

Use the outputTextEncoding property to determines what text conversion to perform when writing text strings to stdout.

outputTextEncoding is only available when running in CGI mode (Server).

*Note:* The naming of the encodings here corresponds directly to the IANA assigned charset names which is why they are perhaps slightly different from text encoding naming elsewhere in the engine.
