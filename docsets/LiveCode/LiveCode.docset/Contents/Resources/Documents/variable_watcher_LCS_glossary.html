Description

Part of the LiveCode development environment, a pane that displays the current values of variables during script execution.

To display the variable watcher, click the variables pane in the scripteditor. To display the variable watcher from the Menubar, choose Debug ? Variables. If you are not debugging a script, this will allow you to watch global variables.
