PathMakeWithSegment

Typeoperator
DictionaryLCB
Syntax
segment path centered at <mCenter> with [ radius <mRadius> | radii <mRadii> ] from <mStartAngle> to <mEndAngle>
Associationscom.livecode.canvas
Summary

Creates a new path.

Parameters
NameTypeDescription
mCenter

An expression which evaluates to a point.

mRadius

An expression which evaluates to a number.

mRadii

An expression which evaluates to a list of numbers.

mStartAngle

An expression which evaluates to a number.

mEndAngle

An expression which evaluates to a number.

Example
// Create an elliptical segment path
variable tPath as Path
put segment path centered at point [100,100] with radii [100,75] from 0 to 270 into tPath
// Create a circular segment path
variable tPath as Path
put segment path centered at point [100,100] with radius 100 from 0 to 270 into tPath
Values
NameTypeDescription
return

A new segment path.

This module specifies the syntax definitions and bindings for canvas drawing operations in modular LiveCode.
