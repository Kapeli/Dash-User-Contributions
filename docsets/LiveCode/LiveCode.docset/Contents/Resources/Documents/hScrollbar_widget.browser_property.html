hScrollbar

Typeproperty
Dictionarywidget.browser
Syntax
set the hScrollbar of <widget> to <pEnabled>
get the hScrollbar of <widget>
Associationscom.livecode.widget.browser
Summary

Controls the visibility of the browser's horizontal scrollbar.

Values
NameTypeDescription
Value

true if the horizontal scrollbar should be enabled and displayed; false otherwise.

RelatedProperty: vScrollbar
Description

This widget displays web content within a native web browser view.

The browser widget can display HTML content generated in LiveCode, or fetch content over the Internet. It supports JavaScript and allows for interaction between LiveCode scripts and JavaScript code.
