The URLEncode function returns the formString, with spaces converted to "+" and characters other than letters and numbers converted to their hexadecimal escape representation.

Description

Use the URLEncode function to encode a URL so it can be safely posted to an HTTP server.

Letters and numbers (alphanumeric characters) are not transformed by the URLEncode function. The representation used for non-alphanumeric characters is a percent sign followed by two hexadecimal digits. For example, the ASCII value of the character ~ is 126; the hexadecimal equivalent of 126 is 7E. So wherever the character ~ a/appears in the formString, it is converted to "%7E".
