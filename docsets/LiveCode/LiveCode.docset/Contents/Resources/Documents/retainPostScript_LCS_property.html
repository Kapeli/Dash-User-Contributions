Specifies whether an EPS object contains the PostScript code.

Values
NameTypeDescription
Value

The retainPostScript of an EPS object is true or false. By default, the retainPostScript property of newly created EPSobjects is set to true.

RelatedProperty: postScript, retainImage
Glossary: EPS, property, object
Description

Use the retainPostScript property to keep or ignore the underlying PostScript code of an EPS object after rendering it.

If an EPS object's retainPostScript property is true, the PostScript code used to draw the object is kept as the object's PostScript property. This means the EPSobject can be printed or scaled and re-drawn without loss of quality.

If the retainPostScript property is false, the PostScript code is deleted once the object has been drawn.

This property is supported only on Unix systems with Display PostScript installed.
