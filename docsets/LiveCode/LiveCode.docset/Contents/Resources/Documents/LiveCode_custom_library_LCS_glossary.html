Description

One of several libraries of custom functions, commands, or properties that come with the LiveCode development environment.

*Important:* If your stacks use commands from a LiveCode custom library, be sure to select the appropriate option in the Inclusions pane of the Standalone Application Settings window to include the library in your standalone.
