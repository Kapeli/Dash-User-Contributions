Description

Operating system that is the underlying core of OS X, written by Apple Computer. Darwin is a variant of Unix.

It is possible to run Darwin by itself, without the Aqua user interface and other services that make up OS X. The Darwin version of the LiveCode engine supports only command-line scripting.
