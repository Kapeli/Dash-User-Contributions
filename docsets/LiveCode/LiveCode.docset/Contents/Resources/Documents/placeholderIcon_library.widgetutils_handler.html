placeholderIcon

Typehandler
Dictionarylibrary.widgetutils
LibraryWidget Utilities
Syntax
placeholderIcon(<pOS>)
Associationscom.livecode.library.widgetutils
Summary

Returns a placeholder icon for a given operating system

Parameters
NameTypeDescription
pOS

The operating system. - "mac" - "ios" - "android"

Description

Use placeholderIcon to fetch an SVG icon that can be used as the placeholder image for a widget that is native to, and therefore only available on, a given operating system.
