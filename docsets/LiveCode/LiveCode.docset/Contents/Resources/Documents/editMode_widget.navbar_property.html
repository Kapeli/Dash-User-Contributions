editMode

Typeproperty
Dictionarywidget.navbar
Syntax
get the editMode of <widget>
set the editMode of <widget> to (true | false)
Associationscom.livecode.widget.navbar
Summary

Whether the widget is in edit mode

RelatedProperty: hilitedItemName, itemArray, itemIcons, itemLabels, itemNames, itemStyle
Message: hiliteChanged
Description

Note: This feature is currently experimental and may change or be removed in future versions.

When the editMode is enabled, the navigation items' icons can be changed by clicking on the outlined regions, and new items can be added with the add button.
