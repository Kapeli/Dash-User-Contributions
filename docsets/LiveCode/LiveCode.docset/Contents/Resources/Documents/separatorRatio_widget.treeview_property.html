separatorRatio

Typeproperty
Dictionarywidget.treeview
Syntax
get the separatorRatio of <widget>
set the separatorRatio of <widget> to {true|false}
Associationscom.livecode.widget.treeview
Summary

Manipulates the size of the arrayData view columns.

RelatedProperty: showSeparator
Description

Use the separatorRatio property to display the keys and values of the arrayData
of the tree view widget in columns separated by a movable divide.

The separatorRatio is the size of the first column as a ratio of the total view area.
