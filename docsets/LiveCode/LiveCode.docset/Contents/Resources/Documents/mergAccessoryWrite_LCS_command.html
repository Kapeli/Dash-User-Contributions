Parameters
NameTypeDescription

The name as returned by mergAccessoryNames()|the protocol as returned by mergAccesoryProtocols(). For example, "p25i|com.bluebamboo.p25i"

pData

The data to write.

pCallcackHandler

The handler to call when the data is written.
