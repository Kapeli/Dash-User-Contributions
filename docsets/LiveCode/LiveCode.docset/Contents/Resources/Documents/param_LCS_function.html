param

Typefunction
DictionaryLCS
Syntax
the param of <parameterNumber>
param(<parameterNumber>)
Summary

Returns the specified parameter passed to the current handler.

EditionCommunity
Introduced1.0
OSmac, windows, linux, ios, android
Platformsdesktop, server, mobile
Parameters
NameTypeDescription
parameterNumber

A non-negative integer.

Example
param(3) -- returns the third parameter
param(0) -- returns the handler name
Values
NameTypeDescription
return

The param function returns the parameter value specified by the parameterNumber. If the parameterNumber is zero, the param function returns the handler name.

RelatedControl structure: function
Glossary: pass, handler, parameter, return
Function: paramCount
Description

LiveCode evaluates the parameters before passing them. So if you call myHandler with the following statement: myHandler 1+1,"A","Hello" && "world" the parameters returned by the param function are 2a/, A, and "Hello World".
