DictionaryLCS

Syntax
mergAWSSimpleDBAddAttribute <pItemsPointer>, <pItemName>, <pAttributeName>, <pAttributeValue>, [<pReplace>]
Associationsmergaws
Summary

Add an attribute to an item in an items object

EditionIndy
Introduced8.0
OSios, mac
Platformsdesktop, mobile
Parameters
NameTypeDescription
pItemsPointer

a pointer to an items object in memory as returned by mergAWSSimpleDBQueryCompleted, mergAWSSimpleDBGetItemCompleted or mergAWSSimpleDBCreateItems

pItemName

an item name

pAttributeName

an attribute name

pAttributeValue

an attribute value

pReplace

(optional with "true" default) if true then when the items are put to the server the value will replace the value for an existing attribute otherwise it will create another attribute of the same name - true - false

Description

mergAWS is an iOS external that adds functions and command for using amazon web services.
