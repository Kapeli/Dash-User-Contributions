foreColor

Typeproperty
Dictionarywidget.navbar
Syntax
get the foreColor of <widget>
set the foreColor of <widget> to <color>
Associationscom.livecode.widget.navbar
Summary

The color of the navigation items that are not highlighted

RelatedProperty: hilitedItemName, itemArray, itemIcons, itemLabels, itemNames, itemStyle
Message: hiliteChanged
Description

The foreColor is used to draw the icons and/or labels of the navigation items that are not currently highlighted.
