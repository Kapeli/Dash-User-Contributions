iconList

Typehandler
Dictionarylibrary.iconsvg
Syntax
iconList()
Associationscom.livecode.library.iconsvg
Summary

Return the available icons in the default icon family.

Values
NameTypeDescription
return

The icon names for which SVG paths are available.

RelatedHandler: iconSVGPathFromName, iconCodepointFromName, iconNames
Description

Return an LCB list of the icon names in the iconsvg library's default icon family.

For each icon name, the SVG path string can be obtained with iconSVGPathFromName or its corresponding codepoint with iconCodepointFromName.
