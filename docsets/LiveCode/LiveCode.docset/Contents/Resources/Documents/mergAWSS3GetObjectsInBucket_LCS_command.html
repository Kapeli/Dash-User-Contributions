mergAWSS3GetObjectsInBucket

Typecommand
DictionaryLCS
Syntax
mergAWSS3GetObjectsInBucket <pBucket>, [<pPrefix>], [<pMarker>], [<pDelimiter>], [<pMaxKeys>]
Associationsmergaws
Summary

Get a list of object keys in a bucket

EditionIndy
Introduced8.0
OSios, mac
Platformsdesktop, mobile
Parameters
NameTypeDescription
pBucket

The name of an existing bucket on S3. A bucket can be thought of as a folder.

pPrefix

(optional with "" default) Limits the response to keys that begin with the specified prefix. You can use prefixes to separate a bucket into different groupings of keys.

pMarker

(optional with "" default) Specifies the key to start with when listing objects in a bucket. Amazon S3 lists objects in alphabetical order.

pDelimiter

(optional with "" default) A delimiter is a character you use to group keys. All keys that contain the same string between the prefix and the first occurrence of the delimiter are grouped under a single result element.

pMaxKeys

(optional with 1000 default) Sets the maximum number of keys returned in the response body. The response might contain fewer keys but will never contain more.

Values
NameTypeDescription
return

An ID for the request to be used by mergAWSS3Cancel
