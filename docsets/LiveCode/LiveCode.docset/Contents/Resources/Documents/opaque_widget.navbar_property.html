opaque

Typeproperty
Dictionarywidget.navbar
Syntax
get the opaque of <widget>
set the opaque of <widget> to {true | false}
Associationscom.livecode.widget.navbar
Summary

Whether the background of the widget is filled

RelatedProperty: backColor
Description

If the opaque property is true, the background of the navigation bar is filled with the backColor. If it is false the background is transparent.
