tab button

DictionaryLCS
Synonymstabbed button,dialog tab,tabbed window,tabbed dialog,tab button
EditionCommunity
RelatedGlossary: button, dialog box
Description

A button type that lets you select a section of a dialog box or window. A tabbed button looks like a stack of file folders, with several tabs across the top. Clicking a tab displays its section in the dialogbox.
