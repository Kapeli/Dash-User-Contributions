editScripts

Typeproperty
DictionaryLCS
Syntax
set the editScripts to {true | false}
Summary

Has no effect and is included in LiveCode for compatibility with imported SuperCard projects.

EditionCommunity
Introduced1.0
OSios, android
Platformsdesktop, server, mobile
RelatedCommand: edit
Glossary: property, SuperCard, LiveCode
Description

The editScripts property is present in LiveCode for compatibility with scripts imported from SuperCard and should not be used in new stacks.

It has no effect on the LiveCode environment, and although it may be set or evaluated without causing an error, its value will always be true.
