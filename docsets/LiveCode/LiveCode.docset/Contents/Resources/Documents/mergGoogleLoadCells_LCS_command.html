Parameters
NameTypeDescription

(optional with "" default) A worksheet identifier as returned by the mergGoogleWorksheetsLoaded message. If empty or no parameter passed then the worksheet will be the currently loaded worksheet.

mergGoogle is an iOS external for interacting with Google services. Currently it only supports Spreadsheets.
