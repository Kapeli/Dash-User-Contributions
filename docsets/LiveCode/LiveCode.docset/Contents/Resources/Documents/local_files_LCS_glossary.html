Synonymslocal file,local files,local disk,local volume,local drive,local folder,local directory,local directories,local system
EditionCommunity
RelatedGlossary: application
Description

A file that's located on the system where the application is running.

A local disk is one that's connected directly to your system, not connected via a local network or the Internet.
