Values
NameTypeDescription
Value

The printerOutput can be one of the following values.The default value depends on the printer driver - Windows Vista uses XPS format. - UNIX uses PostScript format. - Mac OS X uses PDF format. - Older Windows versions use a system-specific format. file depends on the platform: and can occur if you configure an output in the answer printer dialog that LiveCode does not recognise (e.g. WorkFlow output options on Mac OS X 10.4 and above).

device: output to the physical printer
file: <filename> : output to the given file. The format of the
preview: output to print preview (only supported on Mac OS X).
system: output in a system-specific mode. This setting is read-only,
RelatedCommand: open printing, answer printer
Securityprinting
Description

Use the printerOutput property to determine or change the destination of a print job.

Setting the printerOutput to an invalid value will result in a script execution error.

To have an effect, this property must be set before calling openprinting.
