showPict

Typeproperty
DictionaryLCS
Syntax
set the showPict of <card> to {true | false}
Associationscard
Summary

This property is included in LiveCode for compatibility with imported HyperCard stacks.

EditionCommunity
Introduced1.0
Platformsdesktop, server
RelatedKeyword: card, image
Property: visible
Command: show, hide
Glossary: property, LiveCode, HyperCard
Description

In HyperCard, the showPict property determines whether the card image is visible.

In LiveCode, setting this property does not cause a script error, but doing so has no effect.
