request

Typecommand
DictionaryLCS
LibraryLiveCode Script
Syntax
request <expression> {of | from} {program | application} <programAddress>
Summary

The request data gets information from another application via Apple events.

EditionCommunity
Introduced1.0
OSmac
Platformsdesktop, server
Parameters
NameTypeDescription
expression

The string you're asking the other application to evaluate. Its exact format and meaning depends on the other application.

programAddress

The AppleTalk address of the other program. The AppleTalk address consists of three parts, separated by colons: the zone the other computer is in, the name of the computer, and the name of the target program. If the other computer is in the same zone as yours, you can omit the zone. If the other program is running on the same computer, you can omit both the zone and the computer name.

Example
request "the hilite of button 3" from program "HyperCard"
request field "Appname" from program "Ozone:Geek Paradise:FileMaker"
Values

It

If the program supports this Apple event, it evaluates the expression and sends the result back to LiveCode, which places it in the it variable.
