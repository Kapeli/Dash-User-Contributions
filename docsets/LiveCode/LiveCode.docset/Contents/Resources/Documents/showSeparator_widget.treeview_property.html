showSeparator

Typeproperty
Dictionarywidget.treeview
Syntax
get the showSeparator of <widget>
set the showSeparator of <widget> to {true|false}
Associationscom.livecode.widget.treeview
Summary

Whether the separator bar between keys and values is showing or not

RelatedProperty: separatorRatio
Description

Use the showSeparator property to display the keys and values of the arrayData
of the tree view widget in columns separated by a movable divide.
