Description

Handle the deleteAudioclip message if you want to perform cleanup before an audioclip is removed from the stack.

The actual deletion is not triggered by the deleteAudioclip message, so trapping the message and not allowing it to pass does not prevent the audioclip from being removed.

However, the undo command will restore an audioclip after it is deleted by the user.
