checkmark

Typeproperty
DictionaryLCS
Syntax
set the checkmark of <menuItem> to {true | false}
Summary

Has no effect and is included in LiveCode for compatibility with imported HyperCard stacks.

EditionCommunity
Introduced1.0
OSios, android
Platformsdesktop, server, mobile
RelatedKeyword: menuItem
Function: stacks, value
Glossary: property, LiveCode, HyperCard, menu item, error
Description

In HyperCard, the checkmark property determines whether a menuitem has a checkmark next to it.

A handler can set the checkmark to any value without causing a script error, but the actual checkmark is not changed.
