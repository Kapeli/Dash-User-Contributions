dgVisibleLines

Typeproperty
DictionaryDataGrid
LibraryData Grid
Syntax
get the dgVisibleLines of group "DataGrid"
Associationsdatagrid
Summary

Returns the first and last line being displayed in the data grid as a comma delimited list.

Description

Returns the first and last line being displayed in the data grid as a comma delimited list. Useful if you want to provide visual feedback as to which lines are being displayed.
