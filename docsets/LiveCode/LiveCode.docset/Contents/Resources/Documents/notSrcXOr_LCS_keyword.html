Summary

Specifies one of the transfer modes that can be used with the ink property.

Introduced1.0

Platformsdesktop, server, mobile
Example
set the ink of scrollbar "Progress" to notSrcXOr
RelatedKeyword: noOp, notSrcCopy
Glossary: object, property, pixel, operation, Windows, transfer mode, Mac OS, keyword, Unix
Property: ink, pixels
Description

The notSrcXOr mode can be used only on Unix and Windowssystems. On Mac OS systems, objects whose ink property is set to this mode appear as though their ink were set to noOp.

A list of all transfer modes can be found in the transfer mode glossary page for easy reference.
