RelatedFunction: md5Digest, messageDigest
Glossary: byte, function
Description

Compute a message digest of message using the SHA-1 cryptographic hash function.

Warning: Serious flaws have been found in the SHA-1 hash algorithm that make in unsuitable to use for security-critical purposes. Unless you need backwards compatibility with existing systems, use one of the other algorithms available with the messageDigest function
