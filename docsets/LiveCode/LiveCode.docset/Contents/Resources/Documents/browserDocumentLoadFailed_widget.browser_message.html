browserDocumentLoadFailed

Typemessage
Dictionarywidget.browser
LibraryBrowser
Syntax
browserDocumentLoadFailed pUrl, pError
Associationscom.livecode.widget.browser
Summary

Sent when a document has failed to load in the browser.

Parameters
NameTypeDescription
pUrl

The URL of the document.

pError

An error message describing the reason for the failure.

Description

The browserDocumentLoadFailed message is sent to the widget's script object when a new document has failed to load in the browser. The pUrl parameter contains the URL of the document, and the pError parameter gives the reason for the failure.
