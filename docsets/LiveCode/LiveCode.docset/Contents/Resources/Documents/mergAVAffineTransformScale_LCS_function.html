Summary

This will return the affine transform scaled by pScaleX and pScaleY

EditionIndy
Introduced8.0
OSios, mac
Platformsdesktop, mobile
Parameters
NameTypeDescription
pTransform

an affine transform as "a,d,c,d,tx,ty"

pScaleX

the scale to apply in the X direction where 1 makes no chage and 2 doubles the size.

pScaleY

the scale to apply in the Y direction where 1 makes no chage and 2 doubles the size.

Values
NameTypeDescription
return

an affine transform as "a,d,c,d,tx,ty"
