mergAWSProgress

Typemessage
DictionaryLCS
Syntax
mergAWSProgress <pStatus>, <pParam1>, <pParam2>, <pBucket>, <pKey>
Associationsmergaws
Summary

Sent the the LiveCode object that called one of the mergAWSSimpleDB commands

EditionIndy
Introduced8.0
OSios, mac
Platformsdesktop, mobile
Parameters
NameTypeDescription
pStatus

The current status of the request. The status will be one

of

|uploading|downloading|error

pParam1

The value of pParam1 will be different depending on the status. If the status is uploading the value will be the remaining bytes to upload. If it's downloading it will be the bytes received. If it's error then it will be an error message.

pParam2

The value of pParam2 will be different depending on the status. If the status is uploading the total bytes to upload. If it's downloading it will be the total bytes expected to download. Unfortunately SimpleDB seems to send an expected content length of -1 so this parameter should not be relied on.

pBucket

The S3 bucket if the request was an S3 upload or download send with the uploading and downloading status.

pKey

The S3 object key if the request was an S3 upload or download send with the uploading and downloading status.

Description

mergAWS is an iOS external that adds functions and command for using amazon web services.
