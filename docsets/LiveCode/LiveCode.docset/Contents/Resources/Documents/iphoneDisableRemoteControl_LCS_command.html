Summary

EditionCommunity

Example
iphoneDisableRemoteControl
RelatedCommand: iphoneEnableRemoteControl, iphoneSetRemoteControlDisplay, iphoneSetAudioCategory
Function: iphoneRemoteControlEnabled
Message: remoteControlReceived
Description

Use the iphoneDisableRemoteControl command to start stop remote control events when playing audio.

Use the iphoneEnableRemoteControl command to stop receiving remote control events when playing audio.

Note: This functionality is only available on iOS 5.x and above.
