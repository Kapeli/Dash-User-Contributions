BitwiseXor

Typeoperator
DictionaryLCB
Syntax
<Left> bitwise xor <Right>
Associationscom.livecode.bitwise
Summary

Performs a bitwise XOR operation on the binary representations of Left and Right.

Parameters
NameTypeDescription
Left

An expression which evaluates to an integer.

Right

An expression which evaluates to an integer.

Example
variable tVar as Number
put 3 bitwise xor 6 into tVar -- tVar contains 5
Values
NameTypeDescription
return

The integer whose binary representation is the result of the bitwise XOR operation.

Description

Each bit of Left bitwise xor Right is 1 if and only if exactly one of the corresponding bits of the binary representation of Left and that of Right is 1. Otherwise it is 0.
