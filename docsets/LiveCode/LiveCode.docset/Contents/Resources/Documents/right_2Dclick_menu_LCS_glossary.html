EditionCommunity
RelatedGlossary: Unix, Windows, Mac OS, menu item, OS X, menu
Description

A popup menu whose menu items apply to the object or area that the user clicked.

Contextual menus usually appear when you Control-click (on Mac OS and OS X systems) or right-click (on Unix and Windowssystems).
