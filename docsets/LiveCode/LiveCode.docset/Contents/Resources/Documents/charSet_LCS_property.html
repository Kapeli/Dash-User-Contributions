Value

The charSet of a stack is "MacOS" or "ISO". This property is read-only and cannot be set.

RelatedGlossary: property, stack file, Windows, character set, Mac OS, Unix, ISO 8859
Object: stack
Function: numToChar, stacks, platform
Description

Use the charSet property to determine which platform the stack was last saved on.

If the charSet is "MacOS", the stack was last saved on a MacOS system; if the charSet is "ISO", the stack was last saved on a Unix or Windows system.

When you open a stack on a Mac OS system that was last saved on a Unix or Windows system (or vice versa), the text in the stack is translated automatically to the appropriate character set. The process can take a perceptible amount of time, so it's a good idea to save a stack destined for a particular platform on that platform before delivering it to users.

The charSet property is changed for all stacks in the same stackfile when the stack file is saved, so it is not possible for two stacks in the same file to have a different charSet.
