Typeglossary

Syntax
Template Field Editor Properties
Associationsdatagrid
Summary

Datagrid cells can be edited. These property configure that editor.

Description

The template field editor properties are set using the dgTemplateFieldEditor custom property. 

set the dgTemplateFieldEditor["PROPERTY_NAME"] \ 
    of group "Data Grid" to SOME_VALUE
