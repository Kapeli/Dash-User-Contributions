mergGoogleUpdateWorksheet

Typecommand
DictionaryLCS
Syntax
mergGoogleUpdateWorksheet <pTitle>, [<pWorksheet>], [<pColums>], [<pRows>]
Associationsmerggoogle
Summary

Update a given worksheet

EditionIndy
Introduced8.0
OSios, mac
Platformsdesktop, mobile
Parameters
NameTypeDescription
pTitle

A title for the worksheet

pWorksheet

(optional with "" default) A spreadsheet identifier as returned by the mergGoogleWorksheetsLoaded message. If empty or no parameter passed then the worksheet will be the currently loaded worksheet.

pColums

(optional with -1 default) The number of columns in the worksheet. The default value of -1 will leave this value unset.

pRows

(optional with -1 default) The number of rows in the worksheet. The default value of -1 will leave this value unset.
