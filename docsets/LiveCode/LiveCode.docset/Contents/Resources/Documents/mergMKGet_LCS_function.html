Parameters
NameTypeDescription
pProperty
  • rect : a standard rect string in livecode (left,top,right,bottom)
  • visible : true or false
  • center coordinate : a location in decimal coordinates (latitude,longitude)
  • region : the width,height in metres to display within the map
  • shows user location : true or false
  • scroll enabled : true or false

  • user tracking mode : (iOS 5.0+) one of:

    • none
    • follow
    • follow with heading

  • zoom enabled : true or false

  • map type : map type - one of:

    • standard
    • hybrid
    • satellite
Example
put mergMKGet("center coordinate") into tSaveLoc
Values
NameTypeDescription
return

The value of the property.

Description

mergMK is a LiveCode MapKit external that adds a map control which supports showing user location with heading, adding annotation pins and polylines, polygons and circles.

Important: The Map Kit framework uses Google services to provide map data pre-iOS 6. Use of specific classes of this framework (and their associated interfaces) binds you to the Google Maps/Google Earth API terms of service. You can find these terms of service at here.
