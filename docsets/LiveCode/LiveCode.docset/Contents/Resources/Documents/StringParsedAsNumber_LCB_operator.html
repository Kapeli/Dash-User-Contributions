StringParsedAsNumber

Typeoperator
DictionaryLCB
Syntax
<Operand> parsed as number
Associationscom.livecode.arithmetic
Summary

Parses a string as an number

Parameters
NameTypeDescription
Operand

An expression that evaluates to a string.

Example
variable tString as String
variable tNum as optional Number
put "5.6" into tString
put tString parsed as number into tNum -- tNum contains 5.6
variable tNum as optional Number
put "aaaa" parsed as number into tNum


if tNum is nothing then
	put 0 into tNum
end if


// tNum contains 0
Values
NameTypeDescription
return

Operand parsed as a number, or nothing.

Description

Use StringParsedAsNumber when you want to interpret text numerically. nothing is returned if parsing was not possible.
