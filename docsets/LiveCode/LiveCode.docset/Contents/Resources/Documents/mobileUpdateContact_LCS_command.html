mobileUpdateContact

Typecommand
DictionaryLCS
Syntax
mobileUpdateContact <contactArray>, [<title>], [<message>], [<alternateName>]
Summary

Allows the user to create a contact, based on some existing contact data.

EditionCommunity
Introduced5.5.1
OSios, android
Platformsmobile
Parameters
NameTypeDescription
contactArray
title
message
alternateName
(enum): <b>name/</b> Optional Parameters
  • "title": The title that is to appear at the top of the user interface (iOS only).
  • "message": Any message that is to be added to the user interface.
  • "alternateName": An alternate name that is to be displayed in the contact user interface.
Example
put "Just adding this" into tContact["note"]
put "Jobs Ave" into tContact["address"]["home"][1]["street"]
put "Job City" into tContact["address"]["home"][1]["city"]
put "test@livecode.com" into tContactData["email"]["home"][1]
mobileUpdateContact tContact
Values
NameTypeDescription
The result

The ID of the created contact is returned as the result. If no new contact is created, then empty is returned.
