multiSpace

Typeproperty
DictionaryLCS
LibraryLiveCode Script
Syntax
set the multiSpace to {true | false}
Summary

Has no effect and is included in LiveCode for compatibility with imported HyperCard stacks.

EditionCommunity
Introduced1.0
Platformsdesktop, server
RelatedProperty: multiple
Glossary: property, paint tool, LiveCode, HyperCard
Description

In HyperCard, the multiSpace property determines the amount of space left between multiple paint shapes. In LiveCode, the painttools draw a single shape, so the multiSpace property has no effect.
