httpdStop

Typecommand
Dictionaryscript-library.httpd
Libraryhttpd
Syntax
httpdStop <pPorts>
Associationscom.livecode.script-library.httpd
Summary

Stop a HTTP server

Parameters
NameTypeDescription
pPorts

A return delimited list of server ports to stop. If empty all servers are stopped.

Description

Stop all or some HTTP servers by closing the socket and deleting server metadata.
