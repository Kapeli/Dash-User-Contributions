RelatedProperty: lockRecent, lockScreen
Command: show, go
Object: card
Glossary: command, current stack, current card
Description

Use the show cards command to do "flipbook"-style animation or to display a number of cards quickly to the user.

Theform displays number card|cards, starting with the currentcard.

Theform displays all the cards in the current stack, starting and ending with the current card.
