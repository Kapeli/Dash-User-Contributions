Description

Use the sin function to find the sine of an angle.

The sin function repeats every 2*pi radians:

sin(x) = sin(x + 2 * pi)

for any value of x.

The sin function requires its input to be in radians. To provide an angle in degrees, use the following customfunction:

function sinInDegrees angleInDegrees
return sin(angleInDegrees * pi / 180)

end sinInDegrees
