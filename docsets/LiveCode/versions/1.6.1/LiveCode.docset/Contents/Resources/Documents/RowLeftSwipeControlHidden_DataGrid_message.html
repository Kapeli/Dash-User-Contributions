RowLeftSwipeControlHidden

Typemessage
DictionaryDataGrid
LibraryData Grid
Syntax
on RowLeftSwipeControlHidden
Associationsdatagrid,datagrid template custom properties & messages
Summary

Sent when the left hand side data grid swipe control is hidden.

RelatedProperty: enable swipe, left swipe control, control
Command: RowSwipeShowControlForIndexAndSide
Message: RowSwipeHideControl, RowSwipedRight, , RowLeftSwipeControlClicked, , RowRightSwipeControlClicked, , GetLeftSwipeControl, GetRightSwipeControl
Description

RowLeftSwipeControlHidden is sent to your data grid's custom row template when an left hand side swipe control is visible and the user clicks on the data grid, resulting in the swipe control being hidden. Handle RowLeftSwipeControlHidden to perform a custom action on hide.
