showBorder

Typeproperty
Dictionarywidget.headerbar
LibraryHeader Bar
Syntax
get the showBorder of <widget>
set the showBorder of <widget> to {true | false}
Associationscom.livecode.widget.headerbar
Summary

Whether the header bar has a dividing line at the bottom

RelatedProperty: borderColor
Description

If the showBorder property is true, a dividing line at the bottom of the header bar is drawn using the borderColor.
