androidRecorderSetRecordCompressionType

Typehandler
Dictionarylibrary.androidaudiorecorder
LibraryAndroid Audio Recorder
Syntax
androidRecorderSetRecordCompressionType(<pCompressionType>)
Associationscom.livecode.library.androidaudiorecorder
Summary

Set the record compression type

Parameters
NameTypeDescription
pCompressionType

The record compression type - "Default": The default compression type - "AMR NB": AMR Narrowband - "AMR WB": AMR Wideband - "AAC": AAC Low Complexity (AAC-LC) - "HE AAC": High Efficiency AAC (HE-AAC) - "AAC ELD": Enhanced Low Delay AAC (AAC-ELD) - "Vorbis" : Ogg Vorbis

Example
androidRecorderSetRecordCompressionType "AMR NB"
Description

Use the androidRecorderSetRecordCompressionType handler to control the audio encoding used by the audio recorder.
