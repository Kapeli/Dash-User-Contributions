Values
NameTypeDescription
Value

An array describing the marker data

Description

Allows the display of markers at specified points on the map, with associated title and optional subtitle.

The marker array has the following form: { tag for marker: { "title": title of the marker, "subtitle": optional subtitle of the marker, "coordinates": coordinates of the marker }, tag for marker: { ... }, ... }
