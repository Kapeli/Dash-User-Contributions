setProp dgHilite {true | false}
Associationsdatagrid,datagrid template custom properties & messages
Summary

This is a custom property are defined in the behavior script for your template

Description

This property is set by the data grid to set the highlighted state of the row or column control. If the value is "true" then the control should be highlighted. If "false" then it should not be.
