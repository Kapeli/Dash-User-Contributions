Glossary: Standalone Application Settings, standalone application, cursor
Description

Use the iBeam constant to set the cursor to an I-beam, suitable for placing a text insertion point in a field.

The following two statements are equivalent:

set the cursor to iBeam
set the cursor to 9

However, the first is easier to read and understand in LiveCode.

Important: If you use the iBeam cursor or other standard cursors in your application, you must include it when you create your standalone. Make sure the "Cursors" option on the Inclusions section on the General screen of the Standalone Application Settings window is checked.
