cameraControl

Typeobject
DictionaryLCS
LibraryLiveCode Script
Summary

A native camera control that displays a preview image from the current camera device.

Introduced7.1
Example
	cameraControlCreate "myCamera"
	cameraControlCreate "myCamera"
	cameraControlSet "myCamera", "rect", (0, 0, 100, 100)
	cameraControlSet "myCamera", "visible", "true"
	cameraControlSet "myCamera", "device", "back"
RelatedCommand: cameraControlDelete, cameraControlDo, cameraControlCreate, cameraControlSet
Function: cameraControlGet, cameraControlTarget
Description

A cameraControl is created using the cameraControlCreate command
