Summary

Deprecated and will be removed in a future release.

Introduced1.0
OSmac, windows, linux
Platformsdesktop, server
Description

The scriptTextFont property is present in LiveCode for compatibility with scripts imported from HyperCard and should not be used in new code. It will be removed entirely in a future release.
