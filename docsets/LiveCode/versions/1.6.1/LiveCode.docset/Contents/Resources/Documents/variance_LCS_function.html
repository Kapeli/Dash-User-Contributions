A comma-separated list of numbers, or an expression that evaluates to such a list, or an array containing only numbers.

Function: populationVariance, averageDeviation, variance, average, median, populationStandardDeviation, sum, standardDeviation
Description

Use the variance function to analyze the amount of spread in a list of numbers.

The variance function was added in version 6.1 to complement the standardDeviation function. The variance is a measure of how widely distributed the numbers in the numbersList are.
