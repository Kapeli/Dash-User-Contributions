Clock Widget

The clock widget displays the current time in a given time zone, using an analog clock face.

Creating a Clock

A clock widget can be created by dragging it out from the Tools Palette, where it appears with the following icon:

Alternatively it can be created in script using:

create widget as "com.livecode.widget.clock"

Clock Display

When the clock is displaying an AM time, the hands are dark on a light background, and when displaying a PM time, the hands are light on a dark background.

The read-only property isDay also reflects whether the current time is AM or PM.
