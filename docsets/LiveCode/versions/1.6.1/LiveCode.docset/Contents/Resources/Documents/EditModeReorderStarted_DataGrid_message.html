Typemessage
DictionaryDataGrid
LibraryData Grid
Syntax

Summary

Sent when a user starts a dynamic reordering of a data grid.

Parameters
NameTypeDescription
pIndex

The index of the row being reordered.

pLineNo

The line sequence number of the row within the data grid.

RelatedProperty: dgEditMode, edit mode action control, action select control, edit mode reorder control
Command: EditModeShowActionControlForIndex,
Message: EditModeActionSelectControlClicked, EditModeActionControlClicked, EditModeActionControlHidden, GetEditModeActionControl, GetEditModeActionSelectControl, GetEditModeReorderControl
Description

EditModeReorderStarted is sent to your data grid when a user starts a dynamic reordering.
