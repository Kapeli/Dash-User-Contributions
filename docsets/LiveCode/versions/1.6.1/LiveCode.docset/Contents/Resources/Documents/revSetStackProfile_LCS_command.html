Description

Use the revSetStackProfile command to change property settings throughout a stack to a different set of stored settings.

A profile is a set of property settings for an object. You create a profile for an object, and give it a name, using the Property Profiles pane in the object's property inspector. When you change that object to use a profile, the properties you specified in the profile are changed to the settings you specified. This helps you quickly switch the appearance and behavior of the object without needing to set its properties individually.

The revSetStackProfile command changes the current property profile|profile of the specified stack and all the objects in the stack. If an object does not have a property profile|profile with the specified profileName, the object's properties are not changed.

Important: The revSetStackProfile command is part of the Profile library. To ensure that the command works in a standalone application, in the Profiles section on the General screen of the Standalone Application Settings window, make sure you choose to include profiles in your application.
