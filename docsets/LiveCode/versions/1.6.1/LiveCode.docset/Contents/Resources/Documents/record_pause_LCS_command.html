Summary

Pauses the current sound recording.

Introduced6.7
OSmac, windows
Platformsdesktop, server
Example
if the hilite of button "pause" then
   record pause
end if
RelatedCommand: record resume, record sound, stop recording
Property: recording
Description

Use the record pause command to pause a recording being made.

The record pause command pauses a recording started with the recordsound command.

The recording property remains true while a recording is paused.
