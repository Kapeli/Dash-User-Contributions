iconSVGPathFromName

Typehandler
Dictionarylibrary.iconsvg
LibraryIcon SVG Library
Syntax
iconSVGPathFromName(<pName>)
Associationscom.livecode.library.iconsvg
Summary

Returns the SVG path for the given icon.

Parameters
NameTypeDescription
pName

The name of the icon

Values
NameTypeDescription
return

The SVG path to draw the icon with name pName

RelatedHandler: iconNames, iconNamesForFamily
Description

Returns the SVG path for the icon with name pName. Use the iconNames handler to obtain the names of the icons for which this library is able to access SVG paths.

Note: The icon name is accepted in the form "name" or "family/name".
If no family is specified, then the current family is searched first.
If not found, other families are searched for the provided name.
