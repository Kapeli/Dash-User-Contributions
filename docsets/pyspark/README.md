pyspark Docset
=======================

* Author: Ronert Obst (https://twitter.com/ronert_obst)
* Instructions
  * Clone the Apache Spark repo
  * Modify spark/python/docs/conf.py and add `html_theme_options = {"nosidebar": "true"}` under `html_theme = 'nature'`
  * Run make html
  * Then run doc2dash
