Syntax for Indicating a Filename¶

Filename Examples:¶

file.py         => file.py
/tmp/file.py    =>  /tmp/file.py
"C:file.py"     =>  C:file.py  # For Microsoft OS's
'C:file.py'     =>  same as above
'''C:file.py''' =>  same as above
'C:\file.py'    =>  C:\file.py # For Microsoft OS's
'\new.py'       =>  \new.py
'my file.py'    =>  'my file.py'
