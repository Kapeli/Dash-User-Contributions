Phaser Docset
==================

Created by [Colin Ray](https://github.com/rcolinray).

## Generation

To generate the docset, clone [phaser-dash-docset](https://github.com/rcolinray/phaser-dash-docset) and execute the following commands:

```
./setup.sh
./run.sh
```

## Dependencies

- GHC
- Cabal
