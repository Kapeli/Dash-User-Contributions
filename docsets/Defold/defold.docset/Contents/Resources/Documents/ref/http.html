http.request()

perform a HTTP request

Perform a HTTP request.

PARAMETERS

url - target url

method - HTTP method, e.g. GET/PUT/POST/DELETE/...

callback - response callback

[headers] - optional lua-table with custom headers

[post_data] - optional data to send

[options] - optional lua-table with request parameters. Supported entries: 'timeout'= (in seconds)

EXAMPLES

Basic HTTP-GET request. The callback receives a table with the response in the fields status, the response (the data) and headers (a table). 

local function http_result(self, id, response)
    print(response.status)
    print(response.response)
    pprint(response.headers)
end
function example(self)
    http.request("http://www.google.com", "GET", http_result)
end
