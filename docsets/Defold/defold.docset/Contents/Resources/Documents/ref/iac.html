iac.TYPE_INVOCATION

iac type

iac.set_listener()

set iac listener

The listener callback has the following signature: function(self, payload, type) where payload is a table with the iac payload and type is an iac.TYPE_ enumeration.

PARAMETERS

listener - listener callback function (function)

EXAMPLES

Set the iac listener. 

local function iac_listener(self, payload, type)
     if type == iac.TYPE_INVOCATION then
         -- This was an invocation
         print(payload.origin) -- origin may be empty string if it could not be resolved
         print(payload.url)
     end
end

local init(self)
     ...
     iac.set_listener(iac_listener)
end
