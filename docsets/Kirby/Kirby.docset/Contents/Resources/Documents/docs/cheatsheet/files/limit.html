$files->limit($limit)

Returns the collection with a limited set of items

type: $files
text:

Example

<ul>
  <?php foreach($page->files()->limit(3) as $file): ?>
  <li>
    <a href="<?php echo $file->url() ?>">
      <?php echo html($file->filename()) ?>
    </a>
  </li>
  <?php endforeach ?>
</ul>
