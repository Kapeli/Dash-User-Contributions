$page->update($data = array())

Updates the page content with all passed variables.

type: $page
text:

Example

try {

  page('mypage')->update(array(
    'title'        => 'A new title',
    'text'         => 'Some text',
    'anotherfield' => 'Some more data'
  ));

  echo 'The page has been updated';

} catch(Exception $e) {

  echo $e->getMessage();

}
