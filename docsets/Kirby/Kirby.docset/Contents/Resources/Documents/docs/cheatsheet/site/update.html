$site->update($data = array())

Updates all given site variables, which are stored in site.txt

type: $site
text:

Example

try {

  $site->update(array(
    'title'       => 'My Site',
    'description' => 'it is really nice',
  ));

  echo 'The site info has been updated';

} catch(Exception $e) {

  echo $e->getMessage();
