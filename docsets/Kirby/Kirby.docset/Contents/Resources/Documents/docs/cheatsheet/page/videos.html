$page->videos()

Returns a $files collection with all videos in the page directory.

type: $files
text:

Example

<h2>Videos</h2>
<?php foreach($page->videos() as $video): ?>
<video width="300" height="200" controls>
  <source src="<?php echo $video->url() ?>" type="<?php echo $video->mime() ?>">
</video>
<?php endforeach ?>
