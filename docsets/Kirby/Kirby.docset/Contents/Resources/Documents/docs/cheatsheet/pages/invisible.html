$pages->invisible()

Returns all invisible $page objects from the collection

type: $pages
text:

Example

<h2>Invisible subpages</h2>
<ul>
  <?php foreach($page->children()->invisible() as $subpage): ?>
  <li>
    <a href="<?php echo $subpage->url() ?>">
      <?php echo html($subpage->title()) ?>
    </a>
  </li>
  <?php endforeach ?>
</ul>
