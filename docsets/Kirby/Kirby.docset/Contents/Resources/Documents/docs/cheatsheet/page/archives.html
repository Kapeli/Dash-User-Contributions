$page->archives()

Returns a $files collection with all archive files (zip, gzip, tar) in the page's directory

type: $files
text:

Example

<h2>Downloads</h2>
<ul>
  <?php foreach($page->archives() as $archive): ?>
  <li>
    <a href="<?php echo $archive->url() ?>">
      <?php echo $archive->filename() ?>
      (<?php echo $archive->niceSize() ?>)
    </a>
  </li>
  <?php endforeach ?>
</ul>
