$files->offset($offset)

Returns the collection starting from a given offset

type: $files
text:

Example

<ul>
  <?php foreach($page->files()->offset(10) as $file): ?>
  <li>
    <a href="<?php echo $file->url() ?>">
      <?php echo html($file->filename()) ?>
    </a>
  </li>
  <?php endforeach ?>
</ul>
