$site->files()

Returns a $files collection with all files in /content

type: $files
text:

Example

<h2>Files</h2>
<ul>
  <?php foreach($site->files() as $file): ?>
  <li>
    <a href="<?php echo $file->url() ?>">
      <?php echo $file->filename() ?>
      (<?php echo $file->niceSize() ?>)
    </a>
  </li>
  <?php endforeach ?>
</ul>
