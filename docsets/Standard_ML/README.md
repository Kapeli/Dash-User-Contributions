# Standard ML docset

Build requirements:
* `svn` to get html
* `ruby` (with `nokogiri`) to cleanup
* `dashing` to build docset
* `tar` to archive

Build instructions:
```shell
git clone https://github.com/toy/SML-docset-builder.git
cd SML-docset-builder
make
```
