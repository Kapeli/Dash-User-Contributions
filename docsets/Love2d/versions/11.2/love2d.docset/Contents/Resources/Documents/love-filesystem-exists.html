love.filesystem.exists

Deprecated in LÖVE 11.0
This function is deprecated and is replaced by love.filesystem.getInfo.

Check whether a file or directory exists.

Function

Synopsis

exists = love.filesystem.exists( filename )

Arguments

string filename
The path to a potential file or directory.

Returns

boolean exists
True if there is a file or directory with the specified name. False otherwise.
