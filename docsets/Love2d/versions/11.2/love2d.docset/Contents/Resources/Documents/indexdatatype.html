IndexDataType

Available since LÖVE 11.0

Vertex map datatype for Data variant of Mesh:setVertexMap.

Constants

uint16
The vertex map is array of unsigned word (16-bit).
uint32
The vertex map is array of unsigned dword (32-bit).
