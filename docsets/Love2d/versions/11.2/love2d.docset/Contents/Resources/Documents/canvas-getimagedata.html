Canvas:getImageData

Available since LÖVE 0.8.0 and removed in LÖVE 0.10.0
It has been renamed to Canvas:newImageData.

Generates ImageData from the contents of the Canvas. Think of it as taking a screenshot of the hidden screen that is the Canvas.

Function

Synopsis

data = Canvas:getImageData( )

Arguments

None.

Returns

ImageData data
The new ImageData made from the Canvas' image.
