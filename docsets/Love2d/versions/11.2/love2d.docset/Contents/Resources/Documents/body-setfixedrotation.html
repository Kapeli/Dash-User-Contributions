Body:setFixedRotation

Set whether a body has fixed rotation.

Bodies with fixed rotation don't vary the speed at which they rotate. Calling this function causes the mass to be reset.

WARNING: LÖVE 0.6.2 does not yet contain the fix for a bug with this function when it is called with a false argument.

Function

Synopsis

Body:setFixedRotation( isFixed )

Arguments

boolean isFixed
Whether the body should have fixed rotation.

Returns

Nothing.
