love.touch

Available since LÖVE 0.10.0
This module is not supported in earlier versions.

Provides an interface to touch-screen presses.

Functions

love.touch.getPosition Gets the current position of the specified touch-press. 0.10.0
love.touch.getPressure Gets the current pressure of the specified touch-press. 0.10.0
love.touch.getTouches Gets a list of all active touch-presses. 0.10.0
