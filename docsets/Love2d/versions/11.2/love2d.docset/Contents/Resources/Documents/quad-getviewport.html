Quad:getViewport

Gets the current viewport of this Quad.

Function

Synopsis

x, y, w, h = Quad:getViewport( )

Arguments

None.

Returns

number x
The top-left corner along the x-axis.
number y
The top-left corner along the y-axis.
number w
The width of the viewport.
number h
The height of the viewport.
