love.touch.getPressure

Available since LÖVE 0.10.0
This function is not supported in earlier versions.

Gets the current pressure of the specified touch-press.

Function

Synopsis

pressure = love.touch.getPressure( id )

Arguments

light userdata id
The identifier of the touch-press. Use love.touch.getTouches, love.touchpressed, or love.touchmoved to obtain touch id values.

Returns

number pressure
The pressure of the touch-press. Most touch screens aren't pressure sensitive, in which case the pressure will be 1.
