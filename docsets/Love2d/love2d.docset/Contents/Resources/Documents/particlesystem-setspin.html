ParticleSystem:setSpin

Sets the spin of the sprite.

Function

Synopsis

ParticleSystem:setSpin( min, max )

Arguments

number min
The minimum spin (radians per second).
number max (min)
The maximum spin (radians per second).

Returns

Nothing.

Function

Removed in LÖVE 0.9.0
Use ParticleSystem:setSpinVariation instead.

Synopsis

ParticleSystem:setSpin( min, max, variation )

Arguments

number min
The minimum spin (radians per second).
number max (min)
The maximum spin (radians per second).
number variation (0)
The degree of variation (0 meaning no variation and 1 meaning full variation between start and end).

Returns

Nothing.
