Body:getContactList

Available since LÖVE 0.9.2
This method is not supported in earlier versions.

Gets a list of all Contacts attached to the Body.

The list can change multiple times during a World:update call, so some collisions may be missed if the World collide callbacks are not used at all.  

Function

Synopsis

contacts = Body:getContactList( )

Arguments

None.

Returns

table contacts
A list with all contacts associated with the Body.
