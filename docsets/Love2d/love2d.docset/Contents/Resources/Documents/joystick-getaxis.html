Joystick:getAxis

Available since LÖVE 0.9.0
It has been moved from love.joystick.getAxis.

Gets the direction of an axis.

Function

Synopsis

direction = Joystick:getAxis( axis )

Arguments

number axis
The index of the axis to be checked.

Returns

number direction
Current value of the axis.
