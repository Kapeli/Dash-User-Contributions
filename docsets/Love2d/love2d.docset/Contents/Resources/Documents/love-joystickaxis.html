love.joystickaxis

Available since LÖVE 0.9.0
This callback is not supported in earlier versions.

Called when a joystick axis moves.

Function

Synopsis

love.joystickaxis( joystick, axis, value )

Arguments

Joystick joystick
The joystick object.
number axis
The axis number.
number value
The new axis value.

Returns

Nothing.
