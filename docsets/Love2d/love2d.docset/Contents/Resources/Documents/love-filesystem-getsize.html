love.filesystem.getSize

Available since LÖVE 0.9.0
This function is not supported in earlier versions.
Deprecated in LÖVE 11.0
This function is deprecated and is replaced by love.filesystem.getInfo.

Gets the size in bytes of a file.

Function

Synopsis

size, errormsg = love.filesystem.getSize( filename )

Arguments

string filename
The path and name to a file.

Returns

number size
The size in bytes of the file, or nil on failure.
string errormsg (nil)
The error message on failure.
