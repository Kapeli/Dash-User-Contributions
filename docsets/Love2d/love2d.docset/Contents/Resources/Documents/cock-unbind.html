cock.unbind

Unbinds specified map and option from current layout.

Function

Synopsis

cock.unbind ( self, map, option )

Arguments

table self
An object to use.
string map
A control map name to unbind.
string option
Optional. Option to unbind map from. Will revert to defaultOption if not provided.
