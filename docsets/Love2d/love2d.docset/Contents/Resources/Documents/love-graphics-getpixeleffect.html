love.graphics.getPixelEffect

Available since LÖVE 0.8.0 and removed in LÖVE 0.9.0
It has been renamed to love.graphics.getShader.


Returns the current PixelEffect. Returns nil if none is set.

Function

Synopsis

pe = love.graphics.getPixelEffect( )

Arguments

None.

Returns

PixelEffect pe
The current PixelEffect.
