Thread:receive

Available since LÖVE 0.7.0 and removed in LÖVE 0.8.0
It has been renamed to Thread:get.

Receive a message from a thread. Returns nil when a message is not in the message box.

Function

Synopsis

value = Thread:receive(name)

Arguments

string name
The name of the message.

Returns

Variant value
The contents of the message or nil when no message in message box.
